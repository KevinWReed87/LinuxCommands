<h1>Find files with Linux commands</h1>


<h2>Get the current directory information</h2>

- Display your working directory with pwd.
- Display the names of the files and directories in the current working directory with ls.

<img src="https://i.imgur.com/2ucDsSh.png" height="80%" width="80%" alt="Disk Sanitization Steps"/></p>

<h2>Change directory and list the subdirectories</h2>

- Navigate to the /home/analyst/reports directory with cd reports.
- To display the files and subdirectories in the /home/analyst/reports directory use ls.
- The subdirectory in the /home/analyst/reports directory is users.

<img src="https://i.imgur.com/3AA0mhm.png" height="80%" width="80%" alt="Disk Sanitization Steps"/></p>

<h2>Locate and read the contents of a file</h2>

- Navigate to the /home/analyst/reports/users directory using cd users.
- List the files in the current directory with ls
- Display the contents of the Q1_added_users.txt file with cat Q1_added_users.txt
